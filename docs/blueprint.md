# **App Name**: Insight Breast AI

## Core Features:

- Multi-Modal Data Input: Allow users to securely upload mammogram images, or input detailed clinical and pathology data for analysis.
- AI Image Validation & Analysis Tool: Leverage a simulated CNN for mammogram image validation, ensuring input quality, and perform AI-driven analysis to predict malignancy based on visual patterns.
- AI Clinical/Pathology Data Analysis Tool: Utilize AI models to process clinical patient data (age, tumor size, family history) and pathology measurements (radius, area, concavity) to predict breast cancer malignancy.
- Interactive Prediction Summary: Display a clear prediction (Benign/Malignant) along with a confidence score and a visual confidence meter.
- Dynamic Feature Importance: Showcase which input features were most influential in the AI's diagnostic prediction, presented in an easy-to-understand visual format.
- AI-Generated Diagnostic Report Tool: Generate a comprehensive, professional diagnostic report powered by a large language model (e.g., Anthropic's Claude), summarizing findings, recommendations, and disclaimers based on the AI analysis.
- Application Reset: Provide a simple mechanism for users to clear all input data and analysis results, allowing for a fresh start.

## Style Guidelines:

- The primary color is a deep, trustworthy indigo (#402ECC), representing knowledge and professionalism, suitable for interactive elements and prominent text.
- The background color is a very light lavender-gray (#F2F1FA), providing a clean, airy foundation that subtly echoes the primary color without being distracting.
- A vivid sky blue (#33B2FF) serves as the accent color, chosen for its strong contrast with the primary and its association with clarity and progress, used for highlights and calls-to-action.
- The 'Inter' sans-serif font is recommended for both headlines and body text due to its modern, neutral, and highly readable characteristics, which ensure clarity in conveying sensitive medical information.
- Utilize clear and recognizable medical or data-related icons (e.g., upload, brain, checklist) to enhance user comprehension and navigation. Icons should be monochrome and blend with the overall minimalist design.
- Employ a spacious, card-based layout to organize distinct sections of information, promoting visual hierarchy and ease of scanning critical diagnostic details. Consistent padding and margins will contribute to a professional feel.
- Implement subtle, smooth transitions and loading animations, particularly during AI analysis (e.g., spinner with brain icon), to provide feedback and indicate progress without overwhelming the user.
